# 地雷ちゃんパズル ～底辺ぷよぷよ～

底辺キャラクターたちのぷよぷよ風パズルゲーム。  
**ルールは教えない。自分で見つけろ！**

## 🎮 遊び方

1. `index.html` をブラウザで開く（またはGitHub Pagesで公開）
2. キャラクターを選択（数字キー1-5 or クリック）
3. ブロックを操作して…あとは自分で考えろ！

### 操作方法
| キー | アクション |
|------|-----------|
| ← → | 左右移動 |
| ↑ | 回転 |
| ↓ | 高速落下 |
| Space | ハードドロップ |
| P / ESC | ポーズ |

モバイルはタッチボタンで操作可能。

## 🎨 アセット

キャラクターは全てSVGで作成されています。  
`assets/svg/` 内のSVGファイルをPNGに差し替えることで、  
カスタムアートに変更可能です。

## 📁 プロジェクト構成

```
├── index.html          # メインHTML
├── css/
│   └── style.css       # スタイル
├── js/
│   ├── constants.js    # ゲーム定数・設定
│   ├── block.js        # ブロック・ペア定義
│   ├── board.js        # ボード管理・連鎖処理
│   ├── renderer.js     # Canvas描画エンジン
│   ├── input.js        # 入力処理
│   ├── sound.js        # サウンドエフェクト
│   ├── game.js         # メインゲームロジック
│   └── main.js         # エントリーポイント
├── assets/
│   └── svg/            # SVGキャラクターアセット
│       ├── maro.svg
│       ├── amanatsu.svg
│       ├── king.svg
│       ├── shigeru.svg
│       ├── chin.svg
│       ├── kichigai.svg
│       └── stone.svg
└── README.md
```
